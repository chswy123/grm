### 介绍 [Introduction]

基于go+vue的web版redis管理工具【Web redis management tool based on golang and vue】

### 功能清单

- [ ] 管理连接、切换DB

- [ ] 支持 string 类型

- [ ] 支持 list 类型

- [ ] 支持 set 类型

- [ ] 支持 zset 类型

- [ ] 支持 hash 类型

- [ ] 支持 stream 类型

- [ ] SSH 连接 redis

- [ ] 用户登录验证

- [ ] 操作日志
